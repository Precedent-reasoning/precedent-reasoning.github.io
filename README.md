# Precedent Reasoning — Landing Page

This repo hosts the static landing site for **Precedent Reasoning**, an
AI-enabled legal research tool that searches NSW and Commonwealth case law and
explains why retrieved cases matter to the queries. 

Live at
[precedent-reasoning.github.io](https://precedent-reasoning.github.io).

**This repo is a deploy target, not a development repo.** Its contents (aside
from this README) are auto-synced from the `landing/` folder in the
[precedent-reasoning](https://github.com/Precedent-reasoning/precedent-reasoning)
repo via GitHub Actions on every push to `main` — do not contribute to the site files
here directly, as they'll be overwritten on the next sync.

The site includes an interactive demo of the search experience. Results shown
there are illustrative samples only, not real search results. To request a demo with
real search results, please contact us.

The tool provides legal **information**, not legal **advice**.
